# Stainless-ByzCoin Demonstrator

Steps to run locally:

```
make webapp-build
```

Then, in one terminal, run:

```
make webapp-serve
```

And in another terminal:

```
make backend-serve
```

Finally, point your browser to `http://localhost:4200`.
